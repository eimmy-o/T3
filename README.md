# Prolecom - Programming Learning Community

## Project Description
Prolecom is an interactive web platform designed for ITConsultore to encourage collaborative programming learning [2]. The platform is built to strengthen students' technical skills through continuous practice, feedback, and guided mentorship. 

The system integrates the following core modules:
* **Q&A Forum:** A space for structured academic questions, validated by professors and teaching assistants.
* **Programming Challenges:** A bank of practical coding exercises organized by difficulty.
* **Integrated IDE:** An in-browser coding environment that allows students to write, execute, and save code directly.
* **Role Management System:** A hierarchical system handling Students, Teaching Assistants, Professors, Moderators, Support, and Administrators to enforce academic ethics.

## Tech Stack & Engineering Excellence
* **Frontend:** ReactJS 
* **Backend:** PHP 
* **Database:** MySQL 
* **Infrastructure & CI/CD:** Docker, GitHub Actions 
* **Quality Assurance:** SonarQube, ESLint, Unit and Integration Testing
